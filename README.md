Chintal Dress - Official Website

A beautiful, aesthetic, and responsive homepage for Chintal Dress, a clothing brand based in Anand, Gujarat.

Features

Responsive Design: Works on mobile, tablet, and desktop.

Modern UI: Built with Tailwind CSS and Playfair Display typography.

Quick Contact: Direct "Call" links and "Copy Number" functionality.

Sections: Hero, Collections, About, and Footer with Google Maps integration.
